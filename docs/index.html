Configuración única (importante)

YouTube bloquea los servidores de GitHub. Para que funcione, configurá cookies una sola vez:

  1. Extension Chrome: Get cookies.txt LOCALLY
  2. Entrá a youtube.com (logueado) → exportá cookies
  3. PowerShell: [Convert]::ToBase64String([IO.File]::ReadAllBytes("cookies.txt"))
  4. GitHub → Settings → Secrets → Actions
  5. Nuevo secret: YOUTUBE_COOKIES → pegá el base64

Usá una cuenta de Google secundaria. Las cookies caducan cada ~2 semanas.

Los archivos se guardan 7 días como artefacto de GitHub Actions
